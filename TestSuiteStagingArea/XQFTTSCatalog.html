Introduction

The following document is an XSLT transform of the XQuery 1.0 and XPath 2.0 Full-Text test catalog. The test catalog is an XML file containing meta-data about the XQuery interoperability tests published by the FTTF. The meta-data serves to link test queries with their required input sources and results as well as providing further information about the tests (e.g. who the author was, what the purpose of the test is) and categorizing the various tests into their relevant sections. For further information and details about this catalog file, please visit the FTTF homepage at http://www.w3.org/XML/Query/test-suite/.
